Features implemented by Xin Jun Loh
Methods implemented below are to extract data from input.
int getTurn(FILE *fp);
-return 1 or 3 according to input

int getLimit(FILE *fp);
-return -1 if its unlimited, 0 to 9 otherwise

Moves* addMoves(Moves* list, FILE *fp);
-helper method for linking nodes

Moves* getMoves(Moves *list, char src, char dest);
-returns linked list of moves

void getTableau(FILE* fp, Cards tabl[7][20]);
-update 2d array of tableau

Stock* getStock(FILE* fp, Stock* stock);
-return linked list of stock

Waste* getWaste(FILE* fp, Waste* waste);
-return linked list of waste cards

void getFoundation(FILE* fp, char foundation[4][2]);
-return a 2d array of foundation cards

void humanFormat(int valid, int turn, int numMoves, char foundation[4][2], Waste* waste, Cards tabl[7][20]);
-generates outputs according to the human-readable format

void exchangeFormat(int numMoves, int turn, int limit, char foundation[4][2], Stock* stock, Cards tabl[7][20]);
-generates output according to the exchange format

void outFileH_Format(FILE* fp, int valid, int turn, int count, int numMoves, char foundation[4][2], Waste* waste, Cards tabl[7][20]);
-generates output in human-readable format to an output file

void outFileE_Format(FILE* fp, int numMoves, int turn, int limit, char foundation[4][2], Stock* stock, Cards tabl[7][20]);
-generates output in exchange format to an output file

Features implemented by Wei Xian Lim :

Void newMove(char src, char dest, int count);
- input a move and apply the changes to the foundation, waste, stock, or the tableau. 
- returns any error if found. 

Int foudRankCheck(char foud, char input):
- input the given rank of the foundation card and input card and check if the rank is 
  correct.

Int tablRankSuitCheck(char lRank, char lSuit, char rRank, char rSuit):
- input the given rank and suit of the source column and the destination column to check if the rank and suit are correct.
